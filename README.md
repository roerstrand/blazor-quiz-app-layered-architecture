# Blazor Quiz App (.NET)

A quiz application built with **Blazor (.NET)** using a layered architecture.

## 🚀 Features
- AI-generated quiz questions
- User answer handling
- Progress tracking
- Clean UI with component-based structure

## 🧠 Architecture
The application follows a layered approach:
- Presentation (Blazor UI)
- Application logic
- Data / services (API integration)

## 🛠 Tech Stack
- Blazor (.NET)
- C#
- REST API (AI integration)
